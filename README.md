Ferramenta de análise de dados para a Fórmula 1, desenvolvida para transformar métricas complexas em indicadores estratégicos de performance e gestão de recursos.

Funcionalidades:
- Estratégias de Stint: Monitoramento da gestão de pneus e paradas.
- Evolução de Posição: Visualização interativa das disputas e trajetórias em pista.
- Performance: Análise de consistência (tempos de volta) e velocidade máxima.

Tecnologias:
- Python
- FastF1
- Pandas
- Streamlit
- Plotly
